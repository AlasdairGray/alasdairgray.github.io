Big Data Management: Entity Identification Exercise

Author: Alasdair Gray

Exercise

Consider the following statements:

Alasdair is a Lecturer.
Alasdair teaches Big Data Management.
Big Data Management is a course at Heriot-Watt University.
It is taught in Edinburgh and Dubai

Tasks:

  1. Idenitfy the entities and relationships that exist in the statements
    • Entities: Alasdair, …
    • Relationships: isA, …
  2. State which entities need to be represented with nodes (IRIs) and which are data values
  3. What commonsense knowledge/reasoning can you apply to the statements? Give some examples of what you infer.
  4. Draw your entities and relationships as a graph
